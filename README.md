# EOSIO BP rank alarm

The EOSIO BP rank alarm is a project to keep track of your favorite block producer's rank within the 21 elected BP on the EOSIO main net.

* [eos.io](https//:eos.io)
* [eos sw/eden](https://eossweden.se/ )
